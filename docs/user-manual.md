# User Manual

## Login
1. Open the web app on `http://localhost:3000`
2. Enter credentials and sign in

## Employee Actions
- View dashboard attendance summary
- View own leave requests
- Submit new leave request
- View attendance history

## Manager Actions
- All employee actions
- Mark/edit attendance
- Review and approve/reject leave requests
- View reports

## Admin Actions
- Full manager access
- Add/edit/delete employees
- Access all exports and reports

## Status Colors
- Present: green
- Absent: red
- Leave: yellow
